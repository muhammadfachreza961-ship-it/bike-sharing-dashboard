\# 🚲 Bike Sharing Dashboard



\## 📌 Deskripsi

Project ini adalah dashboard interaktif berbasis Streamlit untuk menganalisis data penyewaan sepeda (Bike Sharing Dataset).



Dashboard ini membantu mengeksplorasi pola penyewaan sepeda berdasarkan waktu dan statistik data.



\## 📊 Fitur Dashboard

\- Menampilkan total penyewaan sepeda (cnt)

\- Analisis tren penyewaan berdasarkan waktu

\- Visualisasi data menggunakan bar chart

\- Analisis korelasi antar variabel numerik

\- Dashboard interaktif menggunakan Streamlit

\## ⚙️ Cara Menjalankan



Install dependency:

pip install -r dashboard/requirements.txt



Jalankan Streamlit:

streamlit run dashboard/dashboard.py
